# 公众号流量主限流后推流检查

账号被检测出限流并完成处理后，如何判断是否恢复正常推流？本文通过阅读来源数据分析三种状态。

> 适用范围：仅针对已被检测出**因违规限流**的账号。如果账号未被检测出限流，正常发文即可。

## 操作方法

### 手机端（公众号助手 App）

1. 打开 **公众号助手** → **我的** → **数据统计**
2. **流量分析** → 时间选择 **自定义**
3. 自定义时间设为：**处理完违规问题之后**的时间段
4. 下拉屏幕，找到 **阅读来源和转化分析**

### 电脑端

点击后台首页"昨日阅读人数"的数字，后续步骤参照手机端操作。

## 三种状态判定

将你的阅读来源数据与以下三种状态对比，判断账号恢复情况。

### 状态一：推荐占比 ≥ 90% ✅ —— 完全恢复

- **阅读来源中推荐占比达到 90% 以上**
- 说明账号**已完全恢复正常**，且处于优秀状态
- 按现有节奏正常发文即可

### 状态二：推荐占比 0%~80% 🔄 —— 正在恢复

- 推荐占比在 **0~80% 之间徘徊**
- 说明系统**正在慢慢开始推流**
- 这个阶段需要做：
  - **提升文章质量**，多写优质原创内容
  - **减少不必要的动作**：如互访、串门、与其他作者互动等
  - 耐心等待系统持续推流

### 状态三：推荐占比 = 0% ❌ —— 未恢复

- 推荐占比一直为 **0%**
- 说明系统**还没有开始推流**
- 这个阶段需要做：
  - **立即停止**所有互访、互赞、串门等动作
  - **彻底清理**账号中的违规内容
  - **自己写几篇高质量原创文章**进行发布
  - 目标：先提升到**状态二**（推荐占比从 0 开始回升），再逐步达到**状态一**（推荐占比 90%+）

## 恢复路径

```
状态三 (0%) → 清理违规 + 写高质量原创 → 状态二 (0~80%) → 提升质量 + 减少杂动作 → 状态一 (≥90%)
```

## 注意事项

- 推荐占比是判断账号恢复的核心指标
- 恢复过程中**互访互赞**等动作会干扰系统判断，务必控制
- 处理完违规问题后，需要时间等待系统重新评估，耐心是关键
