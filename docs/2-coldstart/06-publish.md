## 电脑端与手机端发布保姆级流程

为了确保账号权重的统一，我们强烈建议你 **固定一种发布方式**（建议首选电脑网页端发布，操作更精细；不建议今天电脑发布，明天用手机发布，这有利于排版格式稳定）。

### 1. 电脑网页端发布步骤
1. 登录微信公众平台：[mp.weixin.qq.com](https://mp.weixin.qq.com)
2. 在首页点击顶部的 **“新的创作”**，在下拉菜单中选择 **“文章”**。
   - *注意*：起号期选择的是“贴图”（小绿书），写长文时必须选择 **“文章”**。
3. 填入你的**文章标题**（标题是吸引点击的重中之重，后续阶段会详解）。
4. 填写**作者姓名**（直接写你的公众号名字或笔名）。
5. 将在文档里写好、洗好、去 AI 味完的文章正文内容，复制到中央的富文本编辑器中（如果使用了阿一公众号 Markdown 排版工具，就现在工具中排版好，然后点击右上角的“复制”按钮，然后在公众号编辑器中直接粘贴）。
6. 点击右下角 **“发表”** 即可。


![使用阿一公众号markdown排版工具排版后，点击复制按钮](https://n8jmply6oexq.meoo.fun/sb-api/storage/v1/object/public/ayi_oss/public/1780479992667-2-6-2.png)

### 2. 手机端发布步骤（临时备用）
1. 打开并登录 **“公众号助手”** App。
2. 在首页点击 **“新的创作”** 按钮。
3. 选择 **“文章”**，并点击 **“写新文章”**。
4. 依次输入标题和正文，利用底部的功能按钮（`A` 图标）可以对文字加粗、设置字间距或行距。
5. 上传配图并配置好选项后，点击右上角的 **“发表”** 即可。

## 🎨 封面规格设定（极重要）

很多新手在设置封面时胡乱裁剪，导致推送在粉丝列表或者主页时图片被“切头去尾”，非常难看。

公众号要求每一篇文章都要设计 **两个封面**，你需要单独准备：

### 1. 长封面（大封面）
- **推荐尺寸**：`900 × 383` 像素（宽高比为 **2.35 : 1**）。
- **展示位置**：当你的文章推送给关注你的粉丝，在他们的微信订阅号列表里展示的超宽大图。
- **设计心法**：封面必须突出，字体要大且清晰，让人一眼能看出文章核心主题。

### 2. 方封面（小封面）
- **推荐尺寸**：`500 × 500` 像素（宽高比为 **1 : 1**）。
- **展示位置**：当读者点击你的公众号头像进入你的账号主页时，在历史文章列表右侧展示的小正方形配图。
- **设计心法**：尽量选取文章中最具反差、最吸睛的配图，裁剪成正方形即可。

你可以使用创客贴、美图秀秀、稿定设计等工具，创建以上两个对应比例的画布，进行简单的配字配图后直接上传，让整体主页呈现极具高端设计感！

## 2026年4月公众号运营新规
阿一这里还是要提醒一下：2026年4月，公众号平台更新了公众号运营规则，对批量使用AI创作、发布内容的行为进行了专项打击。如果发现大量通过AI制作、发布的低质量内容，将会对账号权限进行降权、封号等措施。
![微信公众号官方发布的运营规范](https://n8jmply6oexq.meoo.fun/sb-api/storage/v1/object/public/ayi_oss/public/1780479992665-2-6-3.png)

所以说现在发布方式也要注意。很多朋友会喜欢那种一键发布到公众号草稿箱的工具，主要通过公众号 API 接口，配合 AI 生成内容，实现批量发布到公众号的操作。之前这个是没有问题的，我之前也做过很多类似的工具，方便用户一键将文章发布到草稿箱。

但是自从有了这个新运营规范以后，批量使用、API 接口发布内容也纳入了监管。所以现在最好手动发布内容，不要大量使用 API 接口来自动发布内容。
