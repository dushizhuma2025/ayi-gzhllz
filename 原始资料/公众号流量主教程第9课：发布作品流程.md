# 公众号流量主教程第9课：发布作品流程

> 讲师：欢少 | 2026年公众号流量主入门基础课程

## 核心观点

发布流程很简单，电脑端和手机端都可。**关键要固定一种发布方式**，不要今天电脑明天手机。

---

## 一、电脑端发布流程

### 步骤

1. 打开公众号后台主页
2. 点击 **"新的创作"** → 选择 **"文章"**
   - ⚠️ 注意：不要选"贴图"（那是小绿书），发文章选"文章"
3. 填写**标题**
4. 填写**作者名字**
5. 填写**正文内容**

### 关键设置项

| 设置项 | 操作 |
|--------|------|
| **封面** | 选文章中的配图作为封面，可裁剪 |
| **原创** | 洗稿检测文皮皮没问题就打开 |
| **赞赏** | 默认不开 |
| **付费** | 不开 |
| **留言** | **默认打开** |
| **合集** | 可不添加（作品归类功能） |
| **原文链接** | 不写 |
| **创作来源** | 不写 |
| **文中广告** | 根据字数和段落自动判断可插几条 |
| **平台推荐** | **默认打开** |

全部完成后点击 **"发表"** → **群发默认打开**即可。

---

## 二、手机端发布流程

### 步骤

1. 打开 **"公众号助手"** App
2. 首页 → 点击 **"新的创作"**
3. 选择 **"文章"**（不要选贴图/视频/音频）
4. 选择 **"写新文章"**
5. 填写**标题**
6. 填写**正文内容**

### 编辑功能区

底部功能栏：
- **A** 按钮 → 设置字体、字号、段落、行距、字间距
- **图片**按钮 → 配图
- 设置**封面**
- 打开**原创**
- 填**作者名**
- **留言**默认打开
- **广告**（广告位能插几条，发时能看见）
- **合集**默认不加
- **平台推荐**默认打开
- **创作来源**默认不写

点击 **"发表"** 即可。

---

## 三、封面规格

公众号有**两个封面**，都需要设置：

### 大封面（长封面）

| 参数 | 值 |
|------|-----|
| 比例 | **2.35 : 1** |
| 推荐尺寸 | **900 × 383 px** |
| 显示位置 | 推送给粉丝时的封面 |

### 小封面（方封面）

| 参数 | 值 |
|------|-----|
| 比例 | **1 : 1** |
| 推荐尺寸 | **500 × 500 px** |
| 显示位置 | 用户点进账号主页时，右侧展示的小图 |

### 封面设计

- **简单方式：** 选文章中的配图，裁剪调整即可
- **精美方式：** 用创客贴、美图秀秀、稿定设计等工具，单独创建画布设计
  - 长封面画布：900 × 383
  - 方封面画布：500 × 500

---

## 四、发布注意事项

| 事项 | 说明 |
|------|------|
| 固定方式 | 电脑或手机，固定一种 |
| 原创开关 | 洗稿检测没问题就打开 |
| 留言 | 默认打开 |
| 平台推荐 | 默认打开，不要关 |
| 封面 | 两个封面都要设置，不能只设一个 |
