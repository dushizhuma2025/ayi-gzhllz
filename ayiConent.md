# ayiConent.md - 公众号流量主教程网站产品逻辑文档

## 项目概述
本项目是一个成体系的“微信公众号流量主实战教程”网页网站。本网站的页面、架构与内容组织结构参考自 OpenCode 网页教程。该教程面向零基础新手，提供从起步建号、快速过门槛、AI高效创作、爆款选题标题设计，到风控防限流的系统化指南。

## 课程体系结构
教程内容划分为以下六大阶段，相关 Markdown 原文位于 `docs/` 目录下：
1. **🚀 快速起步**：涵盖收益逻辑、入池概念、赛道选择、对标账号设定及个人订阅号免费注册流程（应对管理员新规）。
2. **💪 账号起号**：涵盖100粉丝快速过流量主门槛、广告位开通推荐、日记起步法、小绿书图文起号与后台基本排版发布操作。
3. **⚡ 高效创作**：涵盖国内外AI写作工具盘点（豆包、Claude）、核心“倒推指令法”、洗稿相似度（文皮皮）检测与相似度降重、朱雀大模型AI率检测、人工去AI味润色规范。
4. **🎯 爆款运营**：涵盖爆款选题、微信搜一搜筛选低粉爆款标题方法、受众垂直理念及多赛道（中老年、美食、老歌漫画）实战。
5. **🔧 风控避坑**：涵盖5条断流高压线、低创警告一次性删除与申诉话术、限流诊断等风控处理，以及避免收益延迟的财务信息完善指引。
6. **📚 百问百答**：收录常见疑难（如防熟人围观、老号起号可行性、AI自检违规等），并提供违禁词与提示词工具包。

## 技术架构（参考 OpenCode 方案）
- **前端框架**：React 18 + TypeScript
- **构建工具**：Webpack 5 + webpack-dev-server 4.x
- **路由**：react-router-dom 6
- **Markdown 转换**：marked（解析 docs/ 目录下的 markdown 原文）
- **包管理器**：pnpm
- **运行命令**：
  * `pnpm run dev`：启动 Webpack 开发服务器（SPA，数据从自动生成的 JSON 加载）。
  * `pnpm run build`：生产环境打包，并为每个 Markdown 生成独立的静态 HTML 以优化 SEO 与加载速度。

## 已确定的业务逻辑
1. **内容核心**：专注“AI辅助写作 + 人工润色”的低门槛复制路径。
2. **教程组织**：侧边栏菜单结构对应六个阶段，内容由 Markdown 转换生成，支持折叠与展开。
3. **导航与布局**：采用固定顶部导航栏与固定侧边栏的组合结构。采用动态页面匹配器 `ArticleView`，根据当前路由在 `pages.json` 中查找对应的教程正文进行渲染；若访问根路径 `/` 或遇到未定义路由，且当前非 `/1-start/01-intro` 时，将自动重定向至首篇教程 `/1-start/01-intro`。若 `/1-start/01-intro` 本身不存在，则展示友好的错误兜底提示页，避免死循环重定向。
