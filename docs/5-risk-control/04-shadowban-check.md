# 5.3 限流与推流排查诊断

在公众号流量主的实战中，新手遇到阅读量下滑或卡在低位数时，第一反应往往都是：*“我是不是被微信限流了？”*

实际上，微信对账号的推流是有客观客观算法的，真正的限流并不神秘。

本课时将为你系统盘点 **“被限流的 6 种真实情况”**，并教你如何通过官方内置和“URL 黑科技接口”两种方式，直接让系统告诉你你的账号健康状态。

---

## 🔍 限流排查的 6 种常见情况

在自查之前，先对照检查你的账号是否属于以下 6 类被限流的原因：

1. **新号无推荐（非限流，属启动期）**：发了几篇阅读量个位数。这属于新账号还没有进入流量池，需要连续稳定发文 7~10 天让系统学习你的标签。
2. **集中的互粉互阅**：平台反作弊系统检测到你与其他小号有集中互刷点击、串门留言行为，直接砍掉你的账号推荐权重。
3. **内容踩雷违规**：使用了敏感词汇、洗稿不干净、或直接发了 100% 朱雀检测的生硬 AI 水文，触发内容安全风控。
4. **更新无规律，经常断更**：更新频率极不稳定（今天连发 3 篇，接着断更半个月），系统会判定该账号为“非活跃号”，降级推荐资源分配。
5. **排版密密麻麻，完读率极低**：正文不分段、无配图、无粗体，用户点进来秒退。完读率极差导致算法不再给你分配后续流量。
6. **私接未经官方授权的广告**：绕过官方互选广告派单机制私下在正文中插播违规引流营销链接，面临官方直接警告和惩罚。

---

## 🛠️ 两大账号状态检测自救方法

如何直接让官方为我们的账号做出健康度诊断？我们可以采用以下两个方法：

### 方法一：使用官方内置“账号检测”功能
1. **手机端**：打开 **“公众号助手”** App ──> 点击底部的 **“我”** ──> 点击 **“账号检测”** 入口。
2. **电脑端**：登录公众平台网页后台，在左侧栏中寻找 **“账号成长”** ──> 点击 **“账号检测”**。
3. *效果*：如果该功能已对你内测开放，系统会直接显示“全绿对号（正常）”或“黄色感叹号（存在违规低创）”。

### 方法二：提取 ID 拼接 URL 申诉诊断（通用终极法）
很多普通新账号目前没有内置的账号检测功能。你可以使用这个在创客圈内通用的**诊断 URL 提取大法**：

1. 用电脑登录公众号网页后台，观察浏览器地址栏的网址（URL）。
2. 在该 URL 的末尾，通常会包含一串长数字（例如 `https://mp.weixin.qq.com/...&id=12345678`），这串数字就是你公众号的**商户商户商户 ID**。
3. 复制这个 ID，替换到微信官方人工反馈中心用于检测诊断的特定接口 URL 的末尾（替换其中的 `id` 参数）。
4. 在该检测反馈中心弹出的文字框内，填入如下诊断话术模板：
   > “你好，我的公众号最近流量出现严重异常，新发布的原创文章推荐比例直接清零，所有老文章也断崖式断流。辛苦管理员帮我人工检测诊断一下账号最近是否存在违规或AI特征过高的问题，我将积极配合整改。非常感谢您，辛苦了！”
5. 提交之后，在正常工作日，官方客服通常会在 **1 ~ 2 小时内** 给出具体的回复通知（如告知“当前账号状态健康”或者提示“由于AI生成内容占比过高涉嫌低创作被限制推荐”），直接让你看清病因！
