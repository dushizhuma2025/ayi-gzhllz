# 1.1 收益逻辑详解：我们究竟赚的是谁的钱？

## 流量主赚钱的底层逻辑

做项目的第一步，是搞懂钱是从哪里来的。公众号流量主是一个**四方共赢**的商业生态：

| 角色 | 利益点 |
| :--- | :--- |
| **广告商（金主）** | 支付广告费，期望产品/品牌得到曝光与转化。 |
| **腾讯平台** | 收取广告费，提供分发技术和阅读场景，将收益分成给创作者。 |
| **创作者（我们）** | 持续提供优质内容，吸引读者阅读，通过广告展示赚取分成。 |
| **用户（读者）** | 免费阅读有价值、有趣的内容，在合适的时候浏览/点击广告。 |

因为这是一个所有人都能在其中获益的完整闭环，所以，不同于其他“赚完快钱就跑”的项目，它是一个可以**长期、长久**做下去的绿色项目。

## 三大核心广告位详解

开通流量主权限后，系统会自动在你的文章中插入广告。这些广告位主要分为三种：

### 1. 文中广告
*   **展示位置**：文章中间部分。
*   **插入规则**：
    *   文章字数在 **600 字左右**，可插入 **1 条**。
    *   文章字数在 **950 字左右**，可插入 **2 条**（最高上限）。
*   **字数控制秘诀**：最佳的文章长度通常在 **950 ~ 1100 字** 之间。这刚好能插满 2 条文中广告，又不会因篇幅过长影响读者的完读率。

![文中广告案例](/1-1-1.jpg)

### 2. 底部广告
*   **展示位置**：文章正文结束之后、留言区上方。
*   **插入规则**：**无字数要求**。只要开通了流量主并开启了底部广告权限，所有发布的文章都会自动插入。
![底部广告案例](/1-1-2.jpg)

### 3. 留言区广告
*   **展示位置**：展示在评论区（留言区）中间。
*   **插入规则**：当文章的精选留言达到 **4 条以上** 时自动触发。
*   *提示*：这 4 条评论可以由 4 个不同的人各评 1 条，也可以是 2 个人来回互动各评 2 条。
![留言区广告案例](/1-1-3.jpg)

## 核心要点：为什么留言区广告是单价之王？

在三大广告位中，它们的单价（每千次展示或点击的收益分成）通常排序为：

> **留言区广告 > 底部广告 > 文中广告**

### 为什么留言区广告收益最高？
1. **用户行为的精准性**：能滑到最底部并且看留言、参与互动的读者，是完读率最高、最真实的“金牌读者”。广告商最喜欢这类高黏性的用户，因此愿意出更高的曝光和点击价格。
2. **防误触与真实性**：在留言区中展示的广告展示机制更为科学，用户点击的意向度高。

因此，在后续的运营中，**如何引导读者去评论区留言（凑齐 4 条以上的精选留言），是拉升单篇收益的黄金核心技巧之一**。
