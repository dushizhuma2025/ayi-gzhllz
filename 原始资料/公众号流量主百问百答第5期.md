## Q1：起号发图文应该发什么样的类型？

**答案：优先推荐日记起号，图文（贴图）只在特定情况下用。**

| 起号方式 | 起号期流量 | 转型后成功率 | 推荐度 |
|---------|-----------|-------------|--------|
| **日记（文章）** | 几乎不推流（个位数~几十） | ✅ **高**——转型后赛道文章容易推流 | ⭐⭐⭐⭐⭐ |
| 贴图（图文） | ✅ 容易推流 | ❌ **低**——转型后文章基本不推流 | ⭐⭐ |

**什么时候可以用贴图起号？**
只有当你的**赛道文章本身也是贴图类型**时，用同赛道贴图起号效果更好，例如：
- 趣味漫画赛道 → 发7天漫画贴图 → 再转漫画文章
- 星座赛道 → 发7天星座贴图 → 再转星座文章

**如果已经发了一两篇贴图，可以转日记吗？**
可以，直接转成日记继续发就行。

## Q2：垂直是作品垂直，还是针对人群垂直？

**答案：针对受众人群垂直，不是作品垂直。**

很多人以为垂直就是围绕某个赛道创作（比如做退休赛道就只能说退休），这是误区。

正确的理解：**你发的所有内容，看的人都是同一批人，就属于垂直。**

举例：做**中老年退休赛道**，不只可以聊退休——

| ✅ 可以发的内容（受众一致） |
|--------------------------|
| 中老年如何旅游 |
| 中老年如何消磨时光 |
| 中老年如何变得更开心 |
| 中老年如何处理婆媳关系 |
| 中老年鸡汤文 |

只要受众是同一类人群，就可以发。

## Q3：某篇文章突然个位数，其余正常，删还是留？

**判断方法：看它是文章问题还是账号问题。**

### 第一步：判断

| 情况 | 判断 | 处理方式 |
|------|------|---------|
| 明天更新一篇正常推流（几百上千） | ✅ **文章问题**——选题/内容触红线/敏感话题 | 把昨天那篇**删掉**即可 |
| 明天更新还是零/个位数 → 后天还是零 | ❌ **账号问题** | 检测账号 → 反馈 → 申诉 |

### 诊断流程

```
一篇个位数
    ↓
等明天更新一篇新的
    ├─ 新文章正常推流 → 老文章问题 → 删掉老文章
    └─ 新文章还是零 → 再更一篇
         ├─ 第三篇正常 → 第二篇问题 → 删掉
         └─ 第三篇还是零 → 账号问题 → 检测/反馈/申诉/玩贴图
```

## Q4：新号更新两周了一直个位数，怎么办？

**前提：确认选题没问题、内容质量没问题、起号没问题。**

如果这些都确认了，那就是**账号和赛道不匹配**——每个账号有自己"喜欢"的赛道。

**操作建议：**

| 阶段 | 测一个赛道的时间 | 操作 |
|------|----------------|------|
| 首次转型（第一个赛道） | **测10~15天** | 给足时间判断 |
| 后续换赛道 | **测3~5天** | 没推流就换 |
| 重复测试 | — | 直到遇到有推流的赛道为止 |

遇到匹配的赛道后，再垂直更新该赛道的内容。

## Q5：没有账号检测功能，通过网页反馈说账号有不良消息但没说具体哪篇，怎么解决？

**用AI工具帮你分析找出违规文章。**

操作步骤：
1. 打开豆包或其他AI工具
2. 把**历史文章的标题和内容**投喂给AI
3. 把检测出"不良消息"的**界面截图**发给AI
4. 让AI帮你分析**具体哪些点**可能导致不良消息

通过AI分析，就能定位到问题文章，然后删掉它，该反馈反馈，该处理处理。

## Q6：发布前需要检测原创度吗？要的话多少达标？

**分两种情况：**

| 创作方式 | 是否需要检测 | 标准 |
|---------|------------|------|
| **洗稿**（翻改别人的内容） | ✅ **需要检测相似度** | 两篇文章相似度 **≤5%** |
| **AI创作**（给标题/大纲让AI写，或AI给灵感自己组织语言） | ❌ **不需要检测** | — |

洗稿如果相似度超过5%就发，很可能导致原创违规，影响短期流量。

## Q7：老歌赛道，音乐侵权问题怎么找？

**答案：不用担心，侵权概率极低，几乎为零。**

正常找音频、配画面、正常发即可，没有任何问题。

其他被"侵权"的情况（如非商用字体、图片碰瓷）都是专业碰瓷公司搞的，拖到最后都没事。老歌音乐都是正规的，人家也懒得起诉你，没有起诉价值。

## Q8~9：低创作品怎么处理？（删了好像没用 / 更新三天没效果）

**这两个问题很相似，统一回答：**

### 正确的处理流程

1. **找到违规作品**：账号检测界面 → 感叹号 → 点"详情" → 查看被判"低创"的作品
2. **一次性全部删除**（不要今天删两篇、明天删两篇）
3. **去账号检测界面做"账号反馈"**（反馈通过后感叹号不会立刻消失）
4. **优化后续内容**
   - 加**信息增量**：个人看法、观点、心得、感悟
   - 加**生活化内容**、人设定位
   - **不能全文AI**，要有原创性内容
5. **持续更新，等待感叹号消失**

### 感叹号消失周期

| 速度 | 时间 |
|------|------|
| 快 | 一周左右 |
| 慢 | 20~30天甚至更长 |

有真实案例：坚持20多天，感叹号消失当天立马爆了。

**心态建议：** 如果不想频繁注销重开，就好好优化内容挺过这个坎。盘低创说明内容质量不达标，这个过程反而能锻炼创作能力。

## Q10：趣味漫画也会限流吗？以前1000多的浏览量突然没了

**漫画断流通常是因为擦边/低俗。**

自查断流前的几篇作品，大概率是有低俗选题或色情擦边内容。

### 趣味漫画4个注意点

| # | 要点 | 说明 |
|---|------|------|
| 1 | **文案原创度高** | 对话/脚本/标题不能一直照抄对标，容易判低创 |
| 2 | **趣味性要足** | 自己先被逗乐了再发，自己看着都没感觉就别发 |
| 3 | **画风要好看** | 不能太乱太丑，找新颖干净、清晰的画风 |
| 4 | **不要触碰低俗底线** | 色情/低俗内容特别容易断流 |

### 还有一种情况：卡流量池

如果画风和内容都没问题，就是流量卡在某个数值（如卡100、卡1000），坚持更新等待破圈即可。

## Q11：美食一直发文章，是不是也可以发贴图？文章和贴图可以来回切换吗？

**答案：同赛道可以混发，但要以文章为主。**

| 情况 | 可以混发？ |
|------|-----------|
| 同赛道（如美食文章 + 美食贴图） | ✅ **问题不大** |
| 不同赛道 | ❌ 对账号不好 |

**建议：** 如果文章实在不推流了，可以主攻贴图。但文章单价更高，优先文章。

## Q12：新号名字可以改吗？（本来想做老歌，想改做美食）

**答案：不建议改，改名字会降权。**

- 改名字 = 让平台重新审核你的账号，账号又重新开始"认识"你
- **明确会降权**
- 如果是**还没发过作品的新号**，改了还好（空白账号）
- 如果**已经在正常推流了**，**一定不要改**

换赛道其实不需要改名字。除非你强迫症不改就做不下去了，那改也无所谓，但尽量别改。

## Q13：贴图零推荐是什么情况？IP关联的问题吗？

**原因很复杂，可能是：**

| 可能原因 | 说明 |
|---------|------|
| 账号基因不行 | 账号本身有问题 |
| IP没有隔离 | 多账号同IP |
| 设备不干净 | — |
| 贴图本身不容易推流 | 内容类型问题 |
| 贴图是营销内容 | — |
| 资料里有引流信息 | — |
| 素材处理不到位 | — |

**排错方法：**
- 如果排除很多问题还是零推荐
- 发2~3篇零推荐后，换一种**不同风格的贴图类型**再试试
- 另外多账号一定不要用同一个IP，会出问题

## Q14：已经限流了，全是零浏览，怎么办？

**两种情况：**

### 情况一：这个号不想丢

| 选择 | 做法 |
|------|------|
| ✅ **继续做文章** | 检测账号 → 该反馈反馈、该申诉申诉 → 优化后续内容 → 坚持输出 |
| ✅ **转做贴图** | 文章和贴图推荐逻辑不同，流量池完全独立。文章限流**不影响贴图流量** |

贴图流量池跟文章完全独立，即使文章限流了，贴图仍然可以有流量。贴图单价虽低但流量大，整体也还行。

### 情况二：这个号不要了，注销重开

**注销重新注册的注意事项：**
1. **换IP** — 换一个干净的IP
2. **手机上的助手卸载重装**
3. 让网络和设备都更干净
