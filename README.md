## 🚀 项目技术栈与亮点

- **前端架构：** React 18 + TypeScript + React Router 6

- **设计风格：** Apple-inspired 纯享视觉设计，全响应式移动端优先适配，支持侧边栏一键折叠收起。

- **性能优化：** 所有图片和静态资源均托管至高速 OSS 绝对网络 URL 地址，极致秒开体验。

## 🛠️ 本地开发指南

在本地开始运行或二次开发此项目前，请确保您的计算机上已安装 [Node.js](https://nodejs.org/) 环境。

### 1. 安装依赖

在项目根目录下，使用您偏好的包管理器安装所需依赖：

```bash
npm install
# 或者
pnpm install
```

### 2. 启动开发服务器

执行以下指令，脚本将自动编译 docs 目录下的 Markdown 并开启本地热更新开发服务：

```bash
npm run dev
# 或者
pnpm dev
```

启动成功后，可在浏览器中访问：[http://localhost:5173/](http://localhost:5173/)。

### 3. 手动转换文档

如果您修改了 `docs/` 下的任意 Markdown 文件，可以通过以下指令手动将 Markdown 重建为 pages.json 数据源：

```bash
node scripts/convert.js
```

### 4. 生产环境打包

在项目准备上线部署时，运行打包命令编译生成高压缩率的单页静态 HTML 资源：

```bash
npm run build
# 或者
pnpm build
```

编译输出文件将存放在 **`dist/`** 目录中。
