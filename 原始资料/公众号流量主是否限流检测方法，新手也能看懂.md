# 公众号流量主是否限流检测方法，新手也能看懂

很多公众号流量主作者担心账号被限流，本文介绍两种检测账号是否违规限流的方法。

## 方法一：使用内置检测功能（内测中）

目前账号检测功能仍在**内测阶段**，并非所有人都有。部分账号可尝试以下路径：

- **手机端**：公众号助手 App → **我的** → **账号检测**
- **电脑端**：登录公众号后台 → **账号成长** → **账号检测**

如果有该功能，可直接一键检测。

## 方法二：提交检测请求（通用方法）

如果没有内置检测功能，可通过以下 URL 方式提交人工检测：

### 准备工作

1. 在电脑端登录公众号后台
2. 找到公众号后台 URL 末尾的一串数字——这就是**公众号 ID**
3. 将专用的检测接口 URL 中的公众号 ID 替换为自己的 ID

### 提交检测请求

将上面的 ID 替换到位后，打开该 URL，会弹出一个对话框，输入以下内容：

> 你好，我的账号最近流量不正常，不知道哪里违规了，请帮我检查一下账号存在的问题，我这边针对问题进行整改，谢谢。

### 检测结果判断

| 结果 | 说明 | 后续处理 |
|------|------|---------|
| **账号正常** | 无违规问题 | 按现有节奏正常发文即可 |
| **存在异常** | 会提示具体问题（如**AI生成文章低创作度**等） | 见下方处理流程 |

## 出现异常后的处理流程

1. **排查问题文章**：系统只会提示问题类型，不会指明具体是哪篇文章。需要自己根据提示的问题类型，对照已发布的文章进行排查
2. **修改或删除**：将疑似有问题的文章进行修改或删除处理
3. **提交反馈**：处理完成后，点击"**立即继续反馈**"
4. **等待审核**：工作日时间，大约等待 **1~2 小时**收到系统反馈
5. **结果判定**：
   - 审核通过 → **"账号状态恢复正常"**
   - 审核不通过 → 系统会告知不通过的原因，返回第1步继续排查处理，然后重新提交

## 注意事项

- 检测功能内测中，没有内置功能的账号可以通过 URL 方法替代
- 系统提示的问题类型通常是概括性的（如"低创作度"），需要自己逐一排查文章
- 工作日审核效率较高，周末可能延迟
- 反复提交前一定要确认问题已完全解决，否则可能影响账号权重
